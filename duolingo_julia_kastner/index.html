Julia's anagram coding test Reset

This is a tool to check if two phrases (strings) are anagrams.

What's an anagram, you ask? It's whether you can use all the letters in one phrase to make the second phrase without adding or removing letters. For the sake of time, I'm going to ignore spaces, hope that's all right with you.


If you click the "Submit" button, you can see if your strings are anagrams. Give it a try!
